HTML 기본 속성 강의 자료

HTML의 기본 태그와 속성을 알아봅시다!

1. HTML 태그 구조

HTML 태그는 보통 <태그>내용</태그> 형식으로 작성됩니다.

예시 코드:

<p>이것은 단락입니다.</p>

이것은 단락입니다.

2. HTML 기본 속성

HTML 태그에는 다양한 속성을 추가하여 태그의 동작을 제어할 수 있습니다.

공통 속성

  • id: 요소의 고유 식별자를 지정합니다.
  • class: CSS와 JavaScript를 적용하기 위한 클래스 이름을 지정합니다.
  • style: 인라인 스타일을 적용합니다.

예시 코드:

<div id="example" class="box" style="color: red;"> 공통속성 예시 </div>
공통속성 예시

3. 주요 HTML 태그

아래는 자주 사용하는 HTML 태그입니다:

태그 설명
<a> 링크를 만듭니다. href 속성을 사용합니다.
<img> 이미지를 삽입합니다. src와 alt 속성을 사용합니다.
<button> 클릭 가능한 버튼을 만듭니다.
<input> 사용자 입력을 받는 필드를 생성합니다. type 속성을 사용합니다.

4. H 태그 종류

HTML에는 제목을 나타내는 <h1>부터 <h6>까지의 태그가 있습니다.

예시 코드:

<h1>제목 1</h1>
<h2>제목 2</h2>
<h3>제목 3</h3>
<h4>제목 4</h4>
<h5>제목 5</h5>
<h6>제목 6</h6>

제목 1

제목 2

제목 3

제목 4

제목 5
제목 6

5. 이미지 삽입

이미지를 삽입하려면 <img> 태그를 사용합니다.

예시 코드:

<img src="img/image1.jpeg" alt="이미지 설명">

6. 영상 삽입

영상을 삽입하려면 <video> 태그를 사용합니다.

예시 코드:

<video controls>
  <source src="video.mp4" type="video/mp4">
  브라우저가 video 태그를 지원하지 않습니다.
</video>

7. Margin과 Padding

Margin: 요소의 외부 여백을 설정합니다.

Padding: 요소의 내부 여백을 설정합니다.

CSS 예시:

div {
  margin: 50px;
  padding: 10px;
}
여백 설정 예제
테두리 안이 padding
테두리 밖이 margin

8. 수평선 (hr)

수평선을 삽입하려면 <hr> 태그를 사용합니다.

예시 코드:

<hr>

9. 실습하기

아래의 코드를 직접 작성하고 브라우저에서 실행해 보세요:

<!DOCTYPE html>
<html>
<head>
<title>HTML 실습</title>
</head>
<body>
<h1>안녕하세요!</h1>
<p>HTML을 배우고 있습니다.</p>
<hr>
<img src="example.jpg" alt="예제 이미지">
<video controls>
  <source src="example.mp4" type="video/mp4">
</video>
</body>
</html

이 자료는 HTML 학습을 위한 예제입니다.
